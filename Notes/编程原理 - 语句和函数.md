	then

# 2、函数

为了隐藏复杂度，可以把代码打包成“函数”，也叫“方法”或“子程序”

模块化编程不仅可以让单个程序员独立制作APP，也让团队协作可以写更大型的程序

现代编程语言有很多预先写好的函数集合，叫“库”，经过仔细检查，效率高

function expoment(base,exp)
	result = 1
	for i = 1 to exp
		result = result * base
	next
